# Diagram as Code

<details><summary>IBM Cloud Parameters</summary>

1. Diagram
- name
- filename
- folder
- direction = LR or TB
2. Grouping Shapes and Expanded Shapes
- label = primary label (SemiBold font)
- sublabel = secondary label (Regular font)
- direction = LR or TB
3. Collapsed Shapes
- label = primary label (SemiBold font)
- sublabel = secondary label (Regular font)
4. Edge
- label = edge label
- startarrow
- endarrow
- startfill
- endfill
